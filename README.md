# 🎬 Analyzing and Visualizing Movie Ratings

## 📌 Description

This project, **"Analyzing and Visualizing Movie Ratings,"** was developed during my internship at **Hex Software**. The objective of this project is to analyze movie rating datasets (such as IMDb or Rotten Tomatoes) to extract meaningful insights and visualize trends in audience preferences.

The project involves data preprocessing, statistical analysis, and data visualization to better understand rating distributions, top-performing movies, and popular genres.

---

## 🎯 Objectives

* Clean and preprocess raw movie rating data
* Perform statistical analysis on ratings
* Visualize rating distributions using graphs
* Identify top-rated movies and genres
* Build a simple dashboard to present insights

---

## 🚀 Features

* ✔ Data cleaning and handling missing values
* ✔ Calculation of summary statistics (mean, median, mode)
* ✔ Histogram and box plot visualizations
* ✔ Genre-based analysis
* ✔ Identification of highest-rated movies
* ✔ Dashboard for easy interpretation of results

---
